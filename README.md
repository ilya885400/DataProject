Данные репозиторий — это проект для зачёта по дисциплине «Аналитик данных», 3 курс,
Дальневосточный федеральный университет.

Программа создаёт Docker-образ, в котором генерируются искусственные данные
погодной станции для записи в базу данных и их анализа.

Для работы контейнера запустить команду:
`docker compose up`
